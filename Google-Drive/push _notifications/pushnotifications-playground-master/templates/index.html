Push Notifications Playground

The Google Drive API provides push notifications that let you watch for changes to resources. You can use this feature to improve the performance of your application. You can use this feature to eliminate the extra network and computing costs of polling resources to find out if they have changed.

This sample application demonstrates how push notifications in the Google Drive API work. You can use this application to subscribe to all changes to files in your Google Drive, or subscribe to a specific file.

The source code of this application is available at Github.
